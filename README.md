# WiseVision
-------
This is the SRP project at 2016.3.2
Use web make an AR sysem on Android. with OCR function.
the system recognize the content in the picture and then find the key words, then search the key words in the database.
if there is one fit the key words, then it will get the 3D model from the database and show it in the screen in an AR way(based image size to determine the distance from paper, and adjust the attitude with gravity sensor)
